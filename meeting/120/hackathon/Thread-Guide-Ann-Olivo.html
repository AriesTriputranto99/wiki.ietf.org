Thread development recommendations from Ann Olivo.

  1. To get started with OpenThread development, we recommend the EFR32xG24 Explorer Kit.
  2. You'll need to download Simplicity Studio. Additional information on Simplicity Studio can be found in our Simplicity Studio 5 Users Guide.
  3. For more information on OpenThread, please review our OpenThread Quick Start Guide or the OpenThread Github instructions for EFR32.
  4. For those interested in building Matter devices on OpenThread, we encourage you to take a deeper dive into the Matter Quick Start Guide.
