# Sending Announcements

|:------------------------------------------------------------------------:|:-------------------:|:-----------------:|:-------------:|:------------------------:|:------------------:|:----------------:|
| IAB Report                                                               | IAB Chair           | To                | CC            |                          | To                 |                  |
